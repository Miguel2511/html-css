Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B

Nessa frase temos um termo em destaque usando a tag STRONG

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag I

Nessa frase temos um termo em ênfase usando a tag EM

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Texto grande e texto pequeno usando BIG / SMALL.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado usando a tag DEl.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag INS.

Texto sobrescrito

Para inserir coisas do tipo x20+3 use a tag SUP.

Texto subscrito

Para inserir coisas do tipo H2O use a tag SUB.
